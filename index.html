Theory: ‚Ä®Start by establishing the core behavior. Don‚Äôt worry about brand tone, references, or image generation yet. Just ensure the system understands the loop: Image In ‚Üí Context Out.
Iteration 2: The Utility

Adding Features

‚ÄúAllow optional reference images, brand vibe, and style inputs. Extract visible text and key visual details from the product image and include them in the prompt.‚Äù
Theory: Once the base works, layer in utility. Add structured inputs and let the tool enrich the prompt using both user-provided context and inferred image details. Introduce value without adding decision-making complexity.
Iteration 3: The Logic

Adding Iteration & Refinement

‚ÄúAdd a feedback loop where users can critique the output and refine the prompt. Regenerate results based on feedback, with optional image generation.‚Äù
Theory: Logic and iteration are the hardest parts. Introduce refinement only after prompt quality and context extraction are stable. This enables controlled evolution instead of unpredictable outputs.
